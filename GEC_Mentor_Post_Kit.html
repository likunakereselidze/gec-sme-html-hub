Design Specification — For the Canva / Design Team

This section is the full brief for whoever builds the template in Canva. Read all of it before opening Canva. The goal is a premium professional speaker card — not a social media advertisement. Restraint and clarity are the design values.

Canvas & Format

  • Format: 1080 × 1080 px (square — Instagram standard, also works for Facebook)
  • Resolution: 72 dpi for digital, export as PNG
  • Color mode: RGB
  • Safe zone: keep all text within 80px margin from edges
  • No rounded corners on the canvas itself — full bleed square

Brand Colors

#153749 Navy
#226263 Teal
#049978 Green
#F05324 Orange
#F0B91C Gold

Layout Zones (Top → Bottom)

  • Top zone (55%): Circle photo, centered. Diameter: 450 px. Circle has a 4px gold border stroke.
  • GEC logo: top-right corner, 24px from edges. Use GEC wordmark or "GEC" text in gold with a gold rectangle border (2px).
  • Middle band: Mentor name (gold, bold) + title/credential line (teal) + domain tag (white/muted). Center-aligned.
  • Bottom strip (≈ 12% height): Solid teal #226263 background. Program name + date. Center-aligned white text.

Typography

  • Mentor name: Bold, 48–52 px, color: Gold #F0B91C
  • Title/credential: Regular, 26–30 px, color: Teal #6ec4c5 (lighter teal for readability on dark)
  • Domain tag: Light/Regular, 18–20 px, color: White at 55% opacity
  • Bottom strip program name: SemiBold, 22 px, White
  • Bottom strip date: Regular, 17 px, White at 70% opacity
  • Font recommendation: Montserrat (free on Google Fonts — clean, international, works in Georgian/Latin)
  • For Georgian text, Montserrat renders correctly. Alternatively use BPG Arial (Georgian standard) for the Georgian name if needed.

Photo Circle — Execution

  • In Canva: upload mentor's professional headshot
  • Add a circle shape (450 × 450 px), centered at the upper portion of the canvas
  • Right-click photo → "Edit image" → crop to circle, OR use "Frames" → circle frame in Canva
  • Add a circle outline on top: stroke 4px, color Gold #F0B91C, fill: none
  • Photo should be cropped to show face + upper chest (professional headshot style)
  • Avoid photos with busy backgrounds — they compete with the dark navy canvas
  • If mentor has no suitable headshot: use the initials placeholder circle (as shown in the previews below) and request a photo

Bottom Program Strip

  • Strip height: 110–120 px
  • Background: Teal #226263
  • Line 1: GEC SME Business Acceleration Program — SemiBold, 22px, White
  • Line 2: Cohort 1 · June 11, 2026 — Regular, 17px, White 70%
  • Add subtle GEC logo mark (small, white) at left or right of strip if available
  • Separator dot (·) between cohort and date: use a real centered dot, not a hyphen

Design Principles — What Not To Do

  • No emojis anywhere on the frame
  • No gradients on backgrounds — solid navy only
  • No drop shadows on text
  • No decorative borders or patterns — clean and minimal
  • No promotional language on the frame itself (no "Join Now", "Register", "Limited Spots")
  • No clutter — the photo and name are the heroes
  • Consistent frame across all 13 mentors — only the initials circle color changes per mentor (for placeholder); actual headshot replaces this in production

Canva Template Instructions — Step-by-Step for the Design Team

These instructions are written for a team member who has access to Canva Pro (or free). Follow each step in order. Build ONE master frame first, duplicate it 12 times, and only swap the photo + name data per mentor.

Set Up the Canvas

  • Open Canva → click "Create a design"
  • Select "Custom size"
  • Enter: 1080 × 1080 pixels
  • Click "Create new design"
  • Name the file: GEC_Mentor_Frame_MASTER

Set the Background

  • Click the canvas background
  • In the left panel → Background color
  • Enter hex code: #153749 (Navy)
  • This is your base. Every mentor frame uses this same navy background.

Add the GEC Logo (Top-Right Corner)

  • If you have the GEC logo PNG: upload it, place it top-right, 24px from each edge
  • Resize to approximately 100 × 40 px
  • If no logo file: add a Text element with the word GEC, font size 28px, bold, color #F0B91C
  • Draw a rectangle around it: no fill, stroke 2px, color #F0B91C
  • Group the text + rectangle → position top-right at 24px margin

Add the Photo Circle

  • In Canva left panel → Elements → search "Frame" → select the circle frame
  • Resize to 450 × 450 px
  • Position: horizontally centered (X: 315px from left), vertically at approximately Y: 60px from top
  • Upload the mentor's headshot photo → drag and drop it INTO the circle frame
  • Adjust the photo crop so the face is centered and visible
  • Add a circle shape over it: 460 × 460 px, no fill, border stroke 4px, color #F0B91C (gold)
  • Center this outline circle exactly over the photo circle

Add Mentor Name

  • Add a Text element below the photo circle
  • Font: Montserrat Bold (search in Canva font list)
  • Font size: 48–52 px (reduce to 44px if name is long)
  • Color: #F0B91C (Gold)
  • Alignment: Center
  • Position: approximately 30px below the bottom of the photo circle
  • Do not use ALL CAPS — use title case exactly as shown in the mentor list
